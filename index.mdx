## What is Headscale?

Headscale is an open source reimplementation of the Tailscale control server. The control server acts as the coordination point for your Tailscale mesh network — it:

- Exchanges WireGuard public keys between nodes
- Assigns IP addresses from your configured prefix ranges
- Distributes network topology maps to clients
- Enforces access control policies
- Provides DERP relay servers for NAT traversal

With Headscale, you own the control plane. Your data never leaves your infrastructure.

## Quick setup

<Steps>
  <Step title="Install Headscale">
    Install from official packages, container image, or build from source. See [Installation](/setup/install) for all methods.

    ```bash
    # Example: Debian/Ubuntu
    sudo apt install headscale
    ```
  </Step>
  <Step title="Configure and start">
    Copy the example config, set your server URL, and start the service.

    ```bash
    sudo cp /etc/headscale/config.yaml.example /etc/headscale/config.yaml
    # Edit server_url, listen_addr, and database settings
    sudo systemctl enable --now headscale
    ```
  </Step>
  <Step title="Create a user and connect">
    Create a headscale user and register your first node.

    ```bash
    headscale users create alice
    headscale preauthkeys create --user alice
    # On the client:
    tailscale up --login-server https://headscale.example.com --authkey <KEY>
    ```
  </Step>
</Steps>

<Note>
  Headscale requires a publicly reachable HTTPS endpoint on port 443. See [Requirements](/setup/requirements) before installing.
</Note>
